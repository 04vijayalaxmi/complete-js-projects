Section 1 HTML


HTML, or HyperText Markup Language, is a text-based language that describes
the structure of content on a web page. It's the standard markup language for
web pages, and is used by web browsers to display content. HTML is made up of
elements and attributes that mark up the components of a document. These
elements provide the formatting and semantics for the document, such as
creating paragraphs, lists, and tables. HTML elements also label pieces of
content, like "this is a heading" or "this is a link".
